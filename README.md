# modern-cpp-server-boilerplate
Modern C++ boilerplate project for server side. 

# how to run 

Install all dependencies and rebuild.

> cmake -H. -Bbuild
> cmake --build build

## demo description

Provide md5 calculation  via http.

## feature 

- Cmake
- Gcc
- conan
- Jenkins file 
- Docker file 
- Sonar Qube 
- Google Test
- gcov 
- Icov
